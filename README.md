# poswel-website

정적 HTML/CSS/JS 기반 대시보드 데모.

## 실행
- 로컬 : 'index.html'을 브라우저로 열기 또는 VS Code Live Server 사용 또는 임시 주소 사용
- 배포 : Github Pages (Settings → Pages → Branch : 'main', Folder : '/root')

- css
- 'assets/css/app.css' - 스타일

- 'assets/js/auth.js #로그인/로그아웃
